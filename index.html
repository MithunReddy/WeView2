WeView v2

The WeView library is a tool for auto layout of iOS UIs.

  • WeView is an alternative to iOS Auto Layout.
  • WeView aims to provide an easier, more natural, more expressive, and more maintainable way to create dynamic, adaptive UIs with UIKit.
  • WeView takes a declarative approach to layout using familiar concepts like alignment, margins and spacing.
  • WeView builds on iOS’ existing sizing and layout mechanisms.
  • WeView is lightweight: less than 5k lines of code, no dependencies.
  • WeView is compatible with iOS 5 and later.
  • WeView is free and open source. It is available under a permissive license.
